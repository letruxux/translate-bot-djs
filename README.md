# Discord.js translator bot

> literally the simplest bot ever

## Environment variables

Fill out these env vars (using `export`/`set` or via a `.env` file):
```
DISCORD_CLIENT_SECRET=""
DISCORD_CLIENT_ID=""
```

## Install packages
```bash
bun install
```

## Register commands
```bash
bun run ./register.ts
```

## Run the bot
```bash
bun run .
```
